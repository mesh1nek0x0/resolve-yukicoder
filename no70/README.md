## notes
* 最終的に分で出して欲しいみたい
* hとmだけでsまではでない
* :で区切られる
* 寝た時間と起きた時間はスペースで区切られる
* 日付をまたぐ、またがないの判定がある
* 24時間表記で、日付が含まれないため、寝た時間と起きた時間のどっちが大きいかで計算方法が変わる
* 寝た時間が大き場合
 * 日付をまたぐので、寝る時間から24時までの残り時間+起きた時間をそのまま加算
* 起きた時間が大きい場合
 * 日付はまたがないので、起きた時間-寝た時間でよい
* めっちゃ真面目に解いたけど、計算が楽になるように、24時間分起きた方に分で足して、差から24時間ひけばだせる...これは気づかなかった...
