## notes
* これは公式ありそうだけどないんかな？
* 単純にやるなら
 * 総数が奇数ならsortして真ん中の数値を出す
 * 総数が偶数なら中央の2つの数値を足して2.0で除算(少数第1位まで必要)
* 回答後にggってみたけど、↑の考え方自体が公式っぽい

e.g.

5 3 4
これは総数が3で奇数なので、sortして中央の値を出す
3, 4, 5
3 / 2 + 1 = 2 ※配列の2番目の値ですが、要素の参照としては0からはじまるので1番目
ele[N / 2]

1 9 8 4
これは総数が4で偶数なので、sortして中央の2つの平均を出す
1, 4, 8, 9
先頭から数えて(4 / 2)と(4 / 2 + 1)番目の要素を2.0で除算する。
ただし、配列の要素としては0はじまりのため1, 3の参照になるので
(ele[N / 2 - 1] + ele[N / 2]) / 2.0という感じ
