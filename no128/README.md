## notes
* 予算は最低でもn人*1000円が必要なので、未満の場合は0円になる
e.g. 一人k千円をあげるとすると
予算 >= n * (k + 1) * 1000を満たなら、kを1増やしても良い。
予算を超えるようならこれ以上kは増やせない。

24,000で9人パターン

k = 0のとき
  24000 >= 9 * 1 * 1000 = 9000 true
k = 1のとき
  24000 >= 9 * 2 * 1000 = 18000 true
k = 2のとき
 24000 >= 9 * 3 * 1000 = 27000 false

ここでbreak

そんなことしなくてよかった...time overになります。

M人で平等に分けるのでMで割ればいいっすね
24000 / 9 = 2666で切り捨てで、よいですね...。

※1000円単位なので1000で割って端数は切り捨てます
