## notes
* 一番少ない箱からキャンディーをとっていくそうだ
* 空になった箱の数を求める
* N個の箱があってそれぞれCi個ずつ入っていて、全部でM個集めたときおわり
* 一番少ない箱からとっていって、最終的にから箱の数がわかればよい
* sortして少ない順から加算していって、M個を超えたら判定にうつる
 * 残り必要数が0だった場合、から箱は1個増える
 * 残り必要数が負の数だった場合、から箱は増えない
* キャンディーの数が文字列で受け取るので、数値としてsortする必要があるのに注意
