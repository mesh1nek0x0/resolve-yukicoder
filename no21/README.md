## 平均の差

## notes
* 3つ以上のグループにわける
* グループは最低数字を1つ含めば良い
* 差が一番大きなるグループとはminとmaxとそれ以外のグループを作ればよい
* 本質を見抜いて、難しく考えるとダメなやつですね

e.g. 
5 個の数時
3 つのグループ
555
20
432
301
21

min:20
max:555

555 - 20 = 535
