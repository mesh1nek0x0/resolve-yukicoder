## notes
* 一見難しそうに見える気がする
* 二人とも水平方向にしか移動できないため、２点間のX, Yの距離の絶対値を求める
* (diff_x + diff_y) / 2が最短で会える時間
* 差分はabs(a_x - b_x), abs(a_y - b_y)で求まる

e.g.
-12345 6789
1234 -56789

abs(-12345 -1234) = 13579
abs(6789+56789) = 63578

(13579+63578)/2=38578.5
