## notes
* ちと計算が面倒そうな感じ
* Nはゲーム数
* 最終的に求めるのはタイプできた文字数と、ミスった文字数
* 文字数の合計からミスった文字数を減算して出すことになりそう
* floatの計算があったほうがよさげ
* 1000msで12文字タイプできるマン
* 制限時間（Tms）と文字列の組み合わせがN個与えられるので計算してね！って感じ
* 整数に満たないのは切り捨て
* 制限時間750msだと750*12/1000=9なので9文字までOK
* 9文字以上はタイプミス
* 制限時間700msだと700*12/1000=8.4なので切り捨てて8文字までOK
